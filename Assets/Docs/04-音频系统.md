# 音频系统

> `AudioManager` 管理 BGM（背景音乐）和 SFX（音效）的播放。双 AudioSource 架构，BGM 支持循环播放，SFX 支持叠加。

---

## 1. 架构

```
AudioManager (ManagerBase)
├── bgmSource  (AudioSource)  ← BGM 专用，loop = true
└── sfxSource  (AudioSource)  ← SFX 专用，PlayOneShot
```

两个 AudioSource 在场景中挂载到同一个 GameObject 上，通过 Inspector 拖入引用。

---

## 2. 实现

```csharp
public class AudioManager : ManagerBase<AudioManager>
{
    [SerializeField] private AudioSource bgmSource;
    [SerializeField] private AudioSource sfxSource;

    protected override void OnInit()
    {
        if (bgmSource == null) Debug.LogWarning("AudioManager 缺少 bgmSource");
        if (sfxSource == null) Debug.LogWarning("AudioManager 缺少 sfxSource");
    }

    protected override void OnShutdown()
    {
        StopBGM();
    }

    // BGM：切换并循环播放
    public void PlayBGM(AudioClip clip)
    {
        if (clip == null || bgmSource == null) return;
        bgmSource.clip = clip;
        bgmSource.loop = true;
        bgmSource.Play();
    }

    public void StopBGM()
    {
        if (bgmSource != null) bgmSource.Stop();
    }

    // SFX：叠加播放（不打断当前音效）
    public void PlaySFX(AudioClip clip)
    {
        if (clip == null || sfxSource == null) return;
        sfxSource.PlayOneShot(clip);
    }
}
```

---

## 3. BGM vs SFX 技术差异

| | BGM | SFX |
|---|-----|-----|
| 播放方式 | `AudioSource.Play()` | `AudioSource.PlayOneShot()` |
| 循环 | `loop = true` | 不支持（一次性） |
| 叠加 | 同一时刻只有一首 | 可叠加多个 OneShot |
| 停止 | `Stop()` | 无法单独停止 |
| 典型用途 | 场景背景音乐 | UI 点击、攻击、脚步 |

**为什么 SFX 用 PlayOneShot？** 因为战斗中可能多个音效同时触发（多个棋子移动、攻击），PlayOneShot 会自动混合，不会被后续调用打断。

## 4. 使用方式

### 场景中配置

1. 在 AudioManager GameObject 上添加两个 AudioSource 组件
2. 拖入 Inspector 的 `bgmSource` / `sfxSource` 字段
3. AudioClip 文件放在 `Assets/Resources/Audio/` 下

### 代码中调用

```csharp
// 持有 AudioClip 引用的脚本
[SerializeField] private AudioClip buttonClickSFX;
[SerializeField] private AudioClip battleBGM;

// 播放
AudioManager.Instance.PlaySFX(buttonClickSFX);
AudioManager.Instance.PlayBGM(battleBGM);
```

### 通过 ResourceManager 加载

```csharp
AudioClip clip = ResourceManager.Instance.Load<AudioClip>("Audio/Explosion");
AudioManager.Instance.PlaySFX(clip);
```

## 5. 当前局限与扩展方向

| 局限 | 扩展方向 |
|------|---------|
| 仅一个 SFX 源 | 对象池管理多个 SFX 源，支持 3D 空间音效 |
| BGM 直接切换无过渡 | 添加 fade in/out、crossfade |
| 无音量控制 | 添加 Mixer Group，支持全局/分类音量 |
| 无静音 | 添加 `SetMute(bool)` |

## 6. GameFlowManager 中的音频切换

游戏状态切换时伴随 BGM 切换。当前设计（待实现）：

| 状态 | BGM | 说明 |
|------|-----|------|
| OpeningCutscene | Opening | 开场动画配乐 |
| PlayerTurnStart / WaitingForCard / ResolvingCard | Battle | 战斗循环 BGM |
| EnemyTurn | Battle (变奏) | 可考虑切换为紧张版本 |
| LevelComplete | Victory | 胜利号角 |
| Dialogue | 随剧情切换 | 由 DialogueSequence 数据指定 |
| GameOver | GameOver | 失败音乐 |
