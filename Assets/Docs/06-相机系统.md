# 相机系统

> `CameraManager` 基于 Cinemachine 管理多机位切换。通过优先级 (Priority) 机制实现平滑或即时切换。

---

## 1. 设计动机

游戏中有多个视角需求：
- 俯瞰棋盘的策略视角
- 卡牌打出时的特写视角
- 对话时的角色近景
- 桌面互动时的第一人称视角

传统做法是手动移动/旋转 Camera，代码复杂且不直观。Cinemachine 将相机行为抽象为「虚拟相机」，通过切换虚拟相机实现视角切换。

---

## 2. 数据结构

### CameraItem (Frameworks/CameraItem.cs)

```csharp
[System.Serializable]
public class CameraItem
{
    public string name;                           // 相机名（如 "TopDown"）
    public CinemachineVirtualCamera virtualCamera; // 对应的虚拟相机
}
```

### CameraName (Utils/CameraName.cs)

```csharp
public static class CameraName
{
    public static string TOPDOWN   = "TopDown";    // 垂直俯视
    public static string TOPBEHIND = "TopBehind";  // 后方斜 45°
}
```

---

## 3. 核心实现

```csharp
public class CameraManager : ManagerBase<CameraManager>
{
    [SerializeField] private List<CameraItem> camerasList;
    private Dictionary<string, CinemachineVirtualCamera> cameras;

    [SerializeField] private CinemachineVirtualCamera defaultCamera;
    private CinemachineVirtualCamera currentCamera;

    protected override void OnInit()
    {
        ListToDictionary();
        currentCamera = defaultCamera;
        SetCameraPriority();
    }

    // 按名称切换当前相机
    public void SwitchCamera(string cameraName)
    {
        if (!cameras.TryGetValue(cameraName, out var nextCamera))
        {
            Debug.LogWarning("无法找到相机 " + cameraName);
            return;
        }
        currentCamera = nextCamera;
        SetCameraPriority();
    }

    // 所有相机设为 Priority=10，当前相机 Priority=20
    private void SetCameraPriority()
    {
        foreach (var c in cameras.Values)
            c.Priority = 10;
        currentCamera.Priority = 20;
    }

    // Inspector 中填的 List 转 Dictionary（方便查找）
    private void ListToDictionary()
    {
        cameras = new Dictionary<string, CinemachineVirtualCamera>();
        foreach (var c in camerasList)
            cameras[c.name] = c.virtualCamera;
    }
}
```

---

## 4. 优先级机制

Cinemachine 的 `CinemachineBrain` 组件会自动选择场景中 **Priority 最高** 且 **激活** 的虚拟相机作为当前活跃相机。

本系统的做法是：
- 所有虚拟相机默认 Priority = 10
- 当前选中相机 Priority = 20
- 切换时更新优先级，CinemachineBrain 自动过渡

**为什么不用 Enable/Disable？**
- Priority 切换支持平滑 Blend（在 CinemachineBrain 中配置 Blend 曲线）
- Enable/Disable 会导致相机瞬间跳变或重新初始化

---

## 5. 切换流程

```
SwitchCamera("TopDown")
  → cameras.TryGetValue("TopDown")
    → currentCamera = topDownCamera
      → SetCameraPriority()
        → 所有相机 Priority = 10
        → topDownCamera.Priority = 20
          → CinemachineBrain 检测优先级变化
            → 根据 Blend 设置平滑过渡到新相机
```

---

## 6. 使用方式

```csharp
// 切换到俯视相机
CameraManager.Instance.SwitchCamera(CameraName.TOPDOWN);

// 切换到背后相机
CameraManager.Instance.SwitchCamera(CameraName.TOPBEHIND);
```

---

## 7. 场景配置

1. 场景中创建多个 `CinemachineVirtualCamera`
2. 每个配置好 Follow / LookAt / Lens 等参数
3. 在 CameraManager 的 Inspector 中：
   - `camerasList`：依次添加每个相机，填写 name 并拖入对应虚拟相机
   - `defaultCamera`：拖入默认启动的虚拟相机

---

## 8. 已规划相机

| 相机名 | 视角 | 使用场景 |
|--------|------|---------|
| TopDown | 垂直俯视 | 棋盘策略、卡牌选择 |
| TopBehind | 后方斜 45° | 卡牌动效特写、棋子移动动画 |
| TableCloseUp | 桌面近景 | TableInteraction 状态 |
| DialogueCam | 角色对话机位 | Dialogue 状态 |
