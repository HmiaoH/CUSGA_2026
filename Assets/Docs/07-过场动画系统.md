# 过场动画系统

> `CutsceneManager` 负责播放 Timeline 过场动画（如开场背景动画），并在播放结束后执行回调。

---

## 1. 技术基础：Unity Timeline + PlayableDirector

- **Timeline**：Unity 的可视化动画编排工具，在 Timeline 窗口中编排动画轨道
- **PlayableDirector**：播放 Timeline 的组件，挂载在 GameObject 上
- **PlayableAsset**：`.playable` 文件，存储 Timeline 数据

本项目的开场动画文件：`Assets/_Assets/Timeline/OpeningCutscene.playable`

## 3. 核心实现

```csharp
public class CutsceneManager : ManagerBase<CutsceneManager>
{
    [SerializeField] private List<CutsceneItem> cutsceneList;
    private Dictionary<string, PlayableDirector> cutscenes;
    private Action onComplete;

    protected override void OnInit()
    {
        ListToDict();
    }

    public void PlayCutscene(string cutsceneName, Action completeCallback)
    {
        if (!cutscenes.TryGetValue(cutsceneName, out var director))
        {
            Debug.LogWarning("找不到过场动画：" + cutsceneName);
            completeCallback?.Invoke();  // 即使找不到也执行回调（防御性）
            return;
        }

        onComplete = completeCallback;

        // 防止重复订阅
        director.stopped -= OnDirectorStopped;
        director.stopped += OnDirectorStopped;

        director.Play();
    }

    private void OnDirectorStopped(PlayableDirector director)
    {
        director.stopped -= OnDirectorStopped;  // 取消订阅，防止内存泄漏
        onComplete?.Invoke();
        onComplete = null;
    }

    private void ListToDict()
    {
        cutscenes = new Dictionary<string, PlayableDirector>();
        foreach (var c in cutsceneList)
            cutscenes[c.name] = c.director;
    }
}
```

---

## 4. 播放流程

```
PlayCutscene("Opening", callback)
  → cutscenes.TryGetValue("Opening")
    → director.stopped += OnDirectorStopped   // 订阅 stopped 事件
    → director.Play()                          // 开始播放
      → [Timeline 播放中...]
        → Timeline 播放完毕或手动停止
          → director.stopped 事件触发
            → OnDirectorStopped()
              → director.stopped -= OnDirectorStopped  // 取消订阅
              → onComplete?.Invoke()                    // 执行回调
              → onComplete = null
```

## 5. 使用方式

### 播放动画

```csharp
CutsceneManager.Instance.PlayCutscene(CutsceneName.OPENING, () =>
{
    // 动画播放完毕后的操作
    GameFlowManager.Instance.ChangeState(GameState.PlayerTurnStart);
});
```

### 不关心回调

```csharp
CutsceneManager.Instance.PlayCutscene(CutsceneName.OPENING, null);
```

## 6. 场景配置

1. 创建 GameObject，挂载 `PlayableDirector` 组件
2. 将 `.playable` 资产拖入 `Playable` 字段
3. 在 `CutsceneManager` 的 Inspector 中：
   - `cutsceneList`：添加一项，name 填 `"Opening"`，director 拖入上述 PlayableDirector

## 7. 关键设计细节

### 7.1 防御性设计

当找不到指定名称的动画时，`PlayCutscene` 仍会执行 `completeCallback?.Invoke()`。这确保调用方不会因为拼错名称而卡死。

### 7.2 stopped 事件管理

```csharp
director.stopped -= OnDirectorStopped;  // 先移除
director.stopped += OnDirectorStopped;  // 再添加
```

先移除再添加，防止同一个 director 上 `stopped` 事件被重复订阅（如果连续两次调用 `PlayCutscene` 同一个动画）。

### 7.3 回调清空

```csharp
onComplete?.Invoke();
onComplete = null;
```

执行后立即置 null，防止闭包引用导致 GC 延迟。
